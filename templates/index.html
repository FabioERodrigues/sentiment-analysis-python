About This Tool

This web application uses a BERT (Bidirectional Encoder Representations from Transformers) model fine-tuned on movie reviews to analyze sentiment in text data.

The application will:

  • Clean and preprocess your text data
  • Use BERT to classify each text as positive or negative
  • Generate visualizations of the results
  • Allow you to download the analyzed dataset
